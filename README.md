# Quiz Competition 🎯

A simple and interactive Quiz Competition web application built using HTML, CSS, and JavaScript. Users can answer multiple-choice questions, track their score, and view results instantly.

## Live link


https://alokdevops43.github.io/project-/

## 🚀 Features

- Multiple-choice questions
- Instant answer validation
- Score tracking system
- Progress indicator
- Next question navigation
- Final result screen
- Restart quiz option
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

quiz-competition/

## 🎮 How to Use

1. Open the application.
2. Click **Start Quiz**.
3. Select an answer for each question.
4. Move to the next question.
5. View your final score.
6. Restart and play again.
